### 修复 2：视觉模型卡片重新设计

**当前问题**：底部 3 个操作按钮（升级/使用范围/历史版本）是竖排图标+文字的方块网格，拥挤难看。

**新设计**：把 3 个操作收到**卡片右上角的「更多」下拉菜单**（a-dropdown），卡片底部只保留干净的指标区。

新卡片结构：
```
┌──────────────────────────────────┐
│ [图标]  模型名称 ●运行中    [⋯更多] │  头部
│        ● 检测类型                   │
│        📦 v2.3.1 [有新版本]          │

│ │ 📹 18       │ 📋 2        │      │  指标（铺满）

│              [启用/禁用开关 →]      │  底部开关行
```

- 头部右上角：`⋯` 更多按钮 → a-dropdown 弹出菜单（检查升级 / 使用范围 / 历史版本）

**操作菜单项样式**（统一带图标）：
